# FleetTrack

FleetTrack is a modern fleet management and GPS tracking platform built with **Laravel**, **Vue**, **Flutter**, and **Traccar**. The project demonstrates a scalable architecture for managing vehicles, drivers, trips, and real-time location tracking.

This application is being developed as a portfolio project based on a real-world fleet management scenario. It focuses on clean architecture, modern Laravel development practices, API-driven design, and Docker-based development.

## Tech Stack

- Laravel 12
- PHP 8.5 (Laravel Sail)
- Vue.js
- Vite
- MySQL
- Redis
- Mailpit
- Docker (Laravel Sail)
- Flutter (Mobile Application)
- Traccar (GPS Tracking Platform)

## Goals

- Build a professional fleet management system
- Implement real-time GPS tracking
- Manage drivers, vehicles, and trips
- Integrate with Traccar
- Provide a responsive web dashboard
- Develop a cross-platform mobile application
- Follow Laravel best practices and clean architecture
